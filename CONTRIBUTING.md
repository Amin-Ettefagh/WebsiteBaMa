# Contributing

Thanks for contributing to Website Bama.

## Local setup

```bash
npm install
npm run sync:assets
npm run dev
```

## Pull request checklist

- Run `npm run lint`
- Run `npm run build`
- Keep comments in English
- Keep docs in English
- Reuse `lib/site.ts` constants for brand/domain values

## Project conventions

- `legacy/` is the source archive.
- `public/legacy/` is generated by `npm run sync:assets`.
- `public/remote/` is generated by `npm run sync:remote`.
- Contact requests are stored in `data/contact-requests.jsonl`.

## Commit style

Prefer small, focused commits with clear messages.
