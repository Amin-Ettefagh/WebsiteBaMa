راهنمای تغییر Name Server (NS) در سامانه ایرنیک

نسترن غضنفری
24 دی 1404 ساعت 13:00
0

این راهنما یک راهنمای گام به گام برای تغییر Name Server (NS) دامنه در سامانه ایرنیک است تا دامنه شما به سایت فروشگاهی میهن‌شاپ متصل شود. با این کار، می‌توانید دامنه دلخواه خود را به ساخت فروشگاه آنلاین خود متصل کنید و فعالیت فروش اینترنتی خود را به صورت حرفه‌ای ادامه دهید. مراحل ساده و قابل اجرا برای تمامی کاربران طراحی شده است.

راهنمای تغییر Name Server (NS) در سامانه ایرنیک برای ساخت فروشگاه آنلاین با میهن‌شاپ

اگر قصد دارید با استفاده از سیستم فروشگاه ساز میهن‌ شاپ یک  ساخت فروشگاه آنلاین موفق داشته باشید، یکی از مراحل مهم در راهکار فروش اینترنتی، تغییر تنظیمات Name Server (NS) دامنه خود است. این مرحله به شما کمک می‌کند تا دامنه خود را به سایت فروشگاهی خود متصل کنید و محتوای سایت را از طریق دامنه دلخواهتان به مشتریان ارائه دهید.

چرا باید Name Server (NS) را تغییر دهیم؟

وقتی یک دامنه با پسوند .ir خریداری می‌کنید یا قصد دارید دامنه قبلی خود را به ساخت سایت برای فروش محصولات خود در میهن‌شاپ متصل کنید، لازم است Name Server (NS) دامنه خود را به سرویس میهن‌شاپ تغییر دهید.
این کار باعث می‌شود دامنه شما به زیرساخت سایت میهن‌شاپ متصل شود و محتوای سایت از طریق دامنه دلخواهتان نمایش داده شود.
یعنی اینکه:
👉 دامنه‌ تان به سایت شما متصل میشود
👉 سایت شما حرفه‌ای نمایش داده میشود
👉 مشتریان شما به راحتی سایت شما را پیدا می‌کنند

مراحل تغییر Name Server (NS) در سامانه ایرنیک

برای تغییر NS دامنه در سامانه ایرنیک، لطفاً مراحل زیر را دنبال فرمایید:

  1. ابتدا وارد سایت ایرنیک به نشانی ir شوید.

 

2. روی گزینه «ورود» کلیک کرده و با نام کاربری و رمز عبور خود وارد سامانه شوید.

 

 

3. پس از ورود، از بخش «فهرست تمامی دامنه‌ها»، دامنه موردنظر خود را انتخاب کنید.

4. سپس روی گزینه «ویرایش ردیف کارگزاری و نام و میزبانی دامنه» کلیک نمایید.

 

5. در این بخش، NS‌های دامنه را مطابق NSهای ارائه‌شده در پنل کاربری میهن‌شاپ (بخش دامنه‌ها) تنظیم فرمایید. در صورت نیاز می‌توانید از تصویر زیر نیز استفاده نمایید.

 

 

6. پس از وارد کردن NSها، روی گزینه «اعمال» کلیک کنید. در این مرحله DNS دامنه شما به‌روزرسانی خواهد شد.

نکات مهم

  • زمان اعمال تغییرات NS:
    فرآیند اتصال دامنه به میهن‌شاپ از طریق تغییر NS ممکن است بین چند ساعت تا حداکثر ۳ روز کاری طول بکشد. این مدت‌زمان به روند تأیید و بروزرسانی در سامانه ایرنیک بستگی دارد. بنابراین اگر بلافاصله اتصال برقرار نشد، نگران نباشید و کمی صبر کنید تا فرآیند به‌صورت کامل انجام شود.

  • در صورت فراموشی اطلاعات ورود:
    اگر شناسه کاربری، رمز عبور یا پاسخ سوال امنیتی خود را فراموش کرده‌اید، می‌توانید از طریق پشتیبانی سایت ایرنیک پیگیری کنید.

چرا میهن‌شاپ؟

سیستم فروشگاه ساز میهن‌شاپ یکی از بهترین راهکارهای فروش اینترنتی در ایران است. با استفاده از این سیستم، می‌توانید به سادگی یک ساخت فروشگاه آنلاین موفق داشته باشید و محصولات خود را به صورت حرفه‌ای به مشتریان ارائه دهید.
همچنین، این سیستم به شما امکان می‌دهد تا دامنه دلخواه خود را به سایت خود متصل کنید و از طریق آن ساخت سایت برای فروش محصولات خود را به بهترین شکل ممکن ارائه دهید.

نتیجه‌گیری

تغییر Name Server (NS) یک مرحله ضروری در راهکار فروش اینترنتی با استفاده از میهن‌شاپ است. با دنبال کردن مراحل فوق، می‌توانید دامنه خود را به سایت فروشگاهی خود متصل کنید و از طریق آن فعالیت خود را به صورت حرفه‌ای ادامه دهید. اگر هرگونه سوالی در این زمینه داشتید، می‌توانید با پشتیبانی میهن‌شاپ تماس بگیرید.
